## Quick-start

```bash
./phyloFlash.pl -lib LIBRARYNAME -read1 READFILE_F.fq(.gz) -read2 READFILE_R.fq(.gz) [options]
./phyloFlash.pl -help # Help page
./phyloFlash.pl -man # Manual page in pager
```
